Over

Maarten Reynders:

Ik ben Maarten Reynders en graag vertel ik wat meer over mezelf.

Ik ben geboren te Brasschaat op 04/11/1992. Mijn moeder heet Graziëlla en mijn vader heet Peter. Ik heb ook een broer, Ruben.

Van jongs af aan ben ik groot gebracht op de computer door mijn grootvader, die een passie had voor IT. Je moet het dus niet ver zoeken waar ik mijn passie vandaan heb gehaald.
Toen ik ongeveer 14 jaar was heb ik een spel ontdekt genaamd 'Second Life'. Dit is een spel waar de spelers zelf de wereld ontwikkelen. Second life had zijn eigen script taal dat 'LSL' heet. Zo heb ik mijn eerste kennismaking gehad met code. Dit vond ik zo interresant omdat ik door een paar lijnen code iets kon maken dat werkte in Second Life. Het was toen al zichtbaar dat ik een passie creëerde voor programmeren.

Wanneer ik wat ouder was ben ik begonnen met wat te coden in Java met video tutorials. Zo heb ik ook kleine kennismakingen gehad met PHP, html, CSS, javascript enzovoort. Dit was een leuke tijdsbesteding voor mij.

Naarmate ik ouder werd, kwamen de kriebels elke keer terug om programma's te schrijven. Als gevolg heb ik wat games aan het coden geweest zowel van scratch met javascript en java, als met een game engine genaamd 'Unity'.

Uiteindelijk heb ik me het doel voorgenomen om deze hobby en passie als job te gaan doen, dus ben ik begonnen met een graduaatsopleiding bij 'Thomas More'.

Programmeren: Dit is een hobby en een passie van me.
